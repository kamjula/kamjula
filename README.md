## 🚀 Featured Project: CortexOS

CortexOS is a full-stack enterprise data operations platform designed to help teams monitor pipelines, manage datasets, improve data quality, track alerts, and explore AI-assisted workflows from one workspace.

### Core capabilities

- Dataset management with PostgreSQL-backed CRUD operations
- Pipeline monitoring, triggering, retry workflows, and execution logs
- Data quality metrics, validation checks, and issue tracking
- Alerts creation, status management, resolution, and deletion
- AI Copilot interface prototype
- User-management and settings interfaces
- Figma prototype and developer handoff documentation

### CortexOS stack

**Frontend:** React 19 · Vite · JavaScript · Recharts · Lucide React  
**Backend:** Node.js · Express.js  
**Database:** PostgreSQL  
**Tools:** Git · GitHub · VS Code · Figma

[View the CortexOS repository](https://github.com/kamjula/cortex-enterprise)
